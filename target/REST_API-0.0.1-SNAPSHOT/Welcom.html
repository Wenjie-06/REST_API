Instructions for placing/retrieving/paying orders
action method URI
To place an order POST /order/place
To retrieve an order GET /order/display/{email}/{status}
To pay an order PUT /order/pay/{orderID}

An example for an acceptable order:


{"products": [{"productID": "101", "quantity": "100", "unitPrice":"20"}, {"productID": "102", "quantity": "101", "unitPrice":"20"}], "currency": "CAD", "email": "auau77@gmail.com", "postCode":"1234", "countryCode":"0031" }
